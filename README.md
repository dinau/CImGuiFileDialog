### Prerequisites

---

- Windows10 OS or later
- CMake installed 
- GCC compiler installed
- Make command installed

### Build and run

---

```sh
git clone --recurse-submodules https://github.com/dinau/CImGuiFileOpenDialog
```

```sh
cd CImGuiFileOpenDialog
cd examples/glfw_opengl3_fileopendialog
make
```
